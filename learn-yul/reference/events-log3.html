Compute once: keccak256("Transfer(address,address,uint256)") — store as immutable constant in assembly.

log3 anatomy

log3(dataOffset, dataLength, topic0, topic1, topic2)

For ERC20 Transfer:

  • topic0 — event signature hash
  • topic1 — from (indexed address, left-padded to 32 bytes)
  • topic2 — to (indexed address)
  • data — amount (non-indexed uint256, 32 bytes in memory)

Solady transfer emit

mstore(0x20, amount)
log3(0x20, 0x20, _TRANSFER_EVENT_SIGNATURE, caller(), shr(96, mload(0x0c)))

caller() is topic1 (from). shr(96, mload(0x0c)) recovers to from scratch memory after the balance slot was computed — the address still sits at mem[0x0c..0x1f] from the earlier mstore(0x00, to).

Solady approve emit

mstore(0x00, amount)
log3(0x00, 0x20, _APPROVAL_EVENT_SIGNATURE, caller(), shr(96, mload(0x2c)))

mload(0x2c) reads the spender from the allowance scratch layout (spender was stored at mstore(0x20, spender)).
